# amazon-sales-data-analysis
Amazon Sales Data Analysis using Python

Project Overview  
This project analyzes Amazon sales data to understand revenue trends, product performance, and category-wise insights using Python.

Objectives  
- Clean and preprocess sales data  
- Analyze monthly revenue trends  
- Identify top-selling products  
- Understand category-wise revenue contribution  
- Visualize insights using charts  

Tools & Technologies  
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

Key Insights  
- Sales show clear monthly trends  
- Few products generate most revenue  
- Certain categories dominate total sales  
- Peak sales days help business planning  

Dataset  
Amazon sales transaction data in CSV format.

Conclusion  
This project demonstrates practical data analysis skills including data cleaning, trend analysis, and visualization for real-world business scenarios.
